# Dark Calories (식물성 기름의 배신) — 실천 지침 노트

> 저자: Catherine "Cate" Shanahan (케이트 샤나한)
> 사용자가 **건강관리의 바이블**로 삼는 책. @blackpepperpd(후추가루)의 영양 트윗 상당수가
> 이 책을 원전으로 한다는 점이 발췌 대조로 확인됨 → 둘은 사실상 같은 노선.
> 이 노트는 책의 지침을 실천용으로 정리하고, 사용자의 개인 프로토콜을 함께 관리한다.

## ⚑ 노선 확정 (중요 — 혼동 방지)
이 식단은 **포도당 기반(Dark Calories/후추가루/Ray Peat)**이다. **케토(저탄수)가 아니다.**
- ❌ 옛 목표 폐기: "뇌가 당 대신 케톤을 쓰게 한다" → 이건 정반대 학파(케토)의 목표.
  이 노선에선 **케톤 의존 = 포도당 부족 스트레스 상태**라 피하는 대상.
- ✅ 진짜 목표: **시드오일 제거 → 체지방 PUFA↓ → 세포가 포도당을 잘 태움 →
  인슐린 저항성 회복(HOMA-IR<1.0) → 안정 에너지·기분, 포도당으로 잘 굴러가는 뇌.**
- 종착점 = "케톤 대사"가 아니라 **"포도당을 잘 태우는 몸"**. 탄수는 끊지 않고 짝지어 먹는다.
- 두 캠프의 인슐린 저항성 개선법은 정반대: 케토=탄수 끊기 / 우리=PUFA 끊기(탄수 유지). 동시 추구 불가.

## 핵심 명제 (5장 발췌 기준)
1. **씨앗기름 = 염증성 체지방의 원인.** 당분은 "정상 체지방"을 만들지만, 식물성 기름(PUFA)은
   산화가 잘 되는 **"염증성 체지방"**을 만든다. → **시드오일 제거가 최우선 순위.**
2. **인슐린 저항성 = 에너지/대사 문제.** 체지방·세포가 PUFA로 차서 에너지를 제대로 못 만들 때 발생.
   의지·게으름 문제가 아니라 **세포의 연료 처리 문제**(케이트의 "에너지 모형").
3. **혈당은 낮고 안정적으로.** 참조 혈당 약 **65~85 mg/dL**. 식후 급등(스파이크)을 피한다.
4. **혈당 급락 = 산화스트레스.** 단것은 **한 번에 조금씩**, 빈속 폭식 금지.
   PUFA가 빠지면 혈당 곡선 자체가 낮아진다.
5. **순서가 핵심.** "당 끊기"보다 **"시드오일 끊어 당을 깔끔히 태우는 몸"**을 먼저 만든다.
   당은 적이 아니라 연료, 진짜 적은 연료를 못 태우게 하는 PUFA 환경.
6. **악당 서열: 식물성 기름 > 당분.** (Frances Sladek 교수 연구) 미국 당분 소비량이 1999~2016년
   **21% 줄었는데도 비만은 증가** → 당분은 주범이 아님. 시드오일이 산화스트레스·인슐린저항의 진짜 동력.
   당분+시드오일 동시 섭취가 최악 조합이고, **승자(최악)는 시드오일**. → 노력은 **시드오일 제거에 집중**.
   - 근거 실험: 코코넛오일은 **과당을 같이 줘도** 대사·혈당 정상 유지(체지방은 늘어도 "염증성"은 아님).
     대두유는 인슐린저항·제2형 당뇨 신호 유발. 체지방 PUFA는 1900년 이전 ~5% → 현대 >20%.

## 실천 규칙 (Do / Avoid)
### ❌ Avoid (1순위 = 시드오일/PUFA)
- 콩기름, 옥수수유, 해바라기유, 카놀라유, 포도씨유, 면실유, 미강유, "식물성유지" 표기
- 위 기름으로 튀긴 음식(치킨·감자튀김·과자), 시드오일 든 가공식품·소스·드레싱·마요네즈
- 마가린·쇼트닝, 시드오일 함유 가공육

### ✅ Prefer
- **지방:** 버터·기·우지·라드(전통사육)·코코넛오일·올리브유
- **단백질:** 반추동물(소·양) 위주, 달걀, 유제품
- **탄수:** 과일·꿀·잘 익힌 전분 등 — 단, 한 번에 몰아먹지 말고 분산
- **소금 충분히**, 미네랄 동반 수분(과일·우유·코코넛워터)

### 🔁 운영 원칙
- 가공식품은 **원재료명의 "기름 한 줄"로 판정** (팜유/우지=무난, 대두유/식물성유지=기피)
- 외식·라벨 의심되면 사진으로 물어보기 → 이 노트 기준으로 판정

## ⚠️ 균형 장치 (중요)
사용자가 "후추가루보다 더 엄격하게" 가려 한다. 그러나 후추가루 본인도:
- 첫 트윗에서 **극단적 건강법을 "병 걸릴 때까지 하기"라며 풍자/조롱**했고,
- 거의 모든 글에 **"무조건 X은 아니다"라는 자기 단서**를 단다(예: 운동 좌표, 저탄수).
→ 따라서 이 책 노선의 **방향(시드오일 제거)**은 강하게 밀되, **강박적 극단**(완전 무탄수, 영양 불균형,
의학적 치료 대체)은 책·인플루언서의 정신과도 어긋난다. **지속가능한 일관성 > 단기 극단.**
- 실제 질환(당뇨·갑상선 등) 관리 중이면 **이 식이요법이 약·진료를 대체하지 않는다.**

## 개인 프로토콜 (사용자 맞춤)
- **주목표:** 체지방 PUFA 낮추기 (= 시드오일 제거)
- **상태:** 특별한 질환 없음. 단 **에너지 레벨 매우 낮음**, **당뇨전단계 의심**
- **식사환경:** 혼합 (외식/배달 + 집밥)
- **해석:** 저에너지 + 혈당 의심 = 책의 "PUFA로 인한 저대사·인슐린저항" 시나리오와 일치.
  목표(체지방 PUFA↓)가 곧 에너지·혈당 해결의 뿌리.

### 3대 레버 (우선순위)
1. **시드오일 제거 (최우선, 장기전)** — 체지방 리놀레산 반감기 ~1.5~2년 → 효과는 누적·서서히.
   집 기름을 버터/기/우지/코코넛/올리브로 교체. 튀김·가공소스·"식물성유지" 제품 끊기.
2. **혈당 안정 (당뇨전단계 대응)** — 빈속 단독 당분/단 음료 금지. 탄수는 단백질·포화지방과 함께,
   통식품(과일·잘 익힌 전분) 위주로 분산. 스파이크 회피. 목표 공복혈당 감각 65~85.
3. **대사·에너지 끌어올리기** — 아침 햇빛, 소금 충분히, **장시간 단식 금지(코르티솔↑)**,
   코코넛오일/MCT, 탄수 적정 섭취(저탄수로 가지 말 것), 단백질 충분, 수면 우선.

### Anti-crash 식사 전략 (기분장애·혈당 급락 대응)
> 책: 혈당 급락(crash) → 산화스트레스 → 스트레스 호르몬 → 불안·짜증·우울을 "모방". 인슐린 저항이
> 있으면 이 급락이 큼. 사용자는 이유 없는 분노·짜증(가족에게 전이→죄책감) 패턴 있음.
> 핵심: "당이냐 굶음이냐"의 거짓 양자택일을 버리고, **혈당을 급락시키지 않는 안정된 fed state** 유지.
- **굶지 말 것** — 배고픔/브레인포그까지 참으면 스트레스 호르몬 방출. 규칙적으로 먹기.
- **단독 당분 금지** — 단 것만 빈속에 X. 항상 **단백질+포화지방과 함께** → 혈당 완만.
- **끼니 구성** = 단백질 + 좋은 지방(버터/노른자/동물성) + 통식품 탄수 약간.
- 짜증·브레인포그 신호 = "혈당 급락 경보" → 균형 잡힌 끼니/간식으로 빠르게 안정.
- 시드오일 제거로 인슐린 저항이 개선될수록 급락 진폭 자체가 작아짐(근본 해결).

### 약물(GLP-1: 위고비/마운자로) 관련 메모
- 책 입장: GLP-1 약은 food noise·혈당에 도움되나 **뿌리 치료가 아닌 "목발(crutch)"**.
  "몸이 정상이면 필요 없다." → 약을 쓰더라도 **시드오일 0 + anti-crash는 그대로 유지**.
- ⚠️ 처방약. 결정은 **의사와**. 먼저 **HOMA-IR/공복 인슐린/HbA1c**로 baseline 확인 후 상담.
- 기분장애가 반복·심하면 대사 접근과 **동시에** 정신건강 전문가 병행 (둘 중 하나 아님).

### 믿을 수 있는 당분 / 휴대 키트 (요리 귀찮음·투어 중 대응)
> 깨달음: 배고픔 = "뇌의 에너지 부족" → **당(포도당)을 꼭 챙겨야** 함. 버터·고기만 먹으면 허기 안 가심.
> 단, 탄수 **단독은 급락** → 항상 **탄수 + 단백질/지방 한 입**으로 짝지어 "조리 0, 바로" 챙긴다.
- **공식:** 빠른 탄수 + 단백질/지방 = 뇌도 채우고 혈당도 안 꺼지고 배도 든든.
- **① 투어 중·휴대(조리 0):** 🍌바나나+삶은달걀 / 🍠군고구마(khoai lang) / 🌽삶은옥수수 /
  🧀치즈+과일·대추야자 / 🥥코코넛워터(dừa)+달걀·치즈
- **② 집·귀찮을 때(2분):** 🥛전유+과일 ✅ / 요거트(sữa chua)+과일+꿀 / 치즈+과일
- **③ 집 한 끼(5~10분):** 🍳달걀+소금+밥(+과일) ✅ / 계란밥(버터) / 사골 죽·국밥
- **④ 급락 응급:** 🍯꿀 한 스푼 + 우유 한 잔 (꿀 단독 X)
- **베트남 구매 가능:** 군고구마·삶은옥수수·바나나·요거트·코코넛·삶은달걀 = whole food+휴대+시드오일0
- ✅ 사용자가 선호 확정한 조합: **전유+과일**, **달걀+소금+밥**

### 외식·배달 기본 규칙 (혼합 환경)
- 튀김(치킨·감자튀김·튀긴 짜조) → 피하고 **구이·찜·국물** 선택
- 기름 의심되면 사진으로 라벨/메뉴 질문 → 이 노트 기준 판정
- 소스·드레싱·마요 따로 빼기. 베트남 음식은 **튀긴 것보다 쌀국수·구이(분짜)·코코넛 베이스** 우선

### 권장 측정 (책의 혈당 중심과 일치)
- **HOMA-IR (최우선)** — 책이 "인슐린 저항성을 아는 가장 좋은 지표"로 제시. **공복혈당 + 공복 인슐린**
  한 번에 측정 → HOMA-IR = (공복혈당[mg/dL] × 공복인슐린[µU/mL]) / 405. 낮을수록 인슐린 감수성 좋음.
  **목표: < 1.0 (건강/감수성 양호), 3.8↑ = 높음(저항성).** NHANES 2015~2018 기준 미국인 중 1.0 미만은 15%뿐.
  (단순 공복혈당보다 당뇨전단계 초기 포착에 유리)
- **HbA1c**도 함께 보면 좋음. 가정용 혈당계로 공복 65~85 추적 가능.
- 저에너지 원인 감별 위해 **갑상선(TSH/T3)·페리틴(철)** 기본 체크 권장
- ⚠️ 식이요법은 진료·검사를 대체하지 않음. 수치 이상 시 의료진 상담.

## 외식 생존 가이드 (직업=가이드, 외식 빈도 높음)
> 집은 정리됨. 진짜 전장은 **밖**. 베트남/동남아 외식 환경은 **볶음·튀김에 식물성 기름(대두/팜)**을
> 많이 써서, "무엇을 고르냐"가 전부다. 완벽 대신 **구조적으로 기름이 적은 메뉴**를 디폴트로 삼는다.

### 핵심 규칙 한 줄
**"국물 > 구이 > 볶음 > 튀김"** 순서로 고른다. 앞쪽일수록 안전.

### 🟢 고르기 (구조상 시드오일 적음)
- **국물/쌀국수:** 퍼(phở), 분보후에(bún bò Huế) — 뼈 육수 기반, 기름 적음
- **구이(nướng):** 분짜(bún chả), 껌떰(cơm tấm) 숯불 돼지구이, 생선구이, 고기구이
- **찜/조림(kho):** 팟콩(cá kho 등), 코코넛 베이스 조림
- **샤브/훠궈(lẩu):** 육수에 고기·채소 데쳐 먹기 (소스만 조심)
- 흰쌀밥 + 구운 단백질 + 국 조합

### 🔴 피하기 (시드오일 직격탄)
- **튀김 전부:** 짜조(chả giò), 반쎄오(bánh xèo), 넴잔, 튀긴 두부, 프라이드치킨/감자튀김
- **볶음(xào)·볶음밥(cơm chiên):** 웍에 기름 흥건
- **마요네즈·시판 드레싱·반미의 마요** (시드오일 덩어리) → 빼달라고 요청
- 길거리 **재사용 튀김기름** 음식 (산화 PUFA 최악)

### 베트남어 한마디 (현장용)
- "Ít dầu" = 기름 적게 / "Không dầu" = 기름 빼고 / "Không sốt mayonnaise" = 마요 빼고
- "Cho tôi món nướng/luộc" = 구이/삶은 걸로 주세요

### 어쩔 수 없을 때 (손님 접대·선택권 없음)
- **가장 안 튀긴 메뉴**를 고르고, **소스는 따로**. 죄책감은 버린다.
- 그날 나머지 끼니를 깨끗이 해서 **하루 총 PUFA**를 관리. (체지방 PUFA는 장기전 — 한 끼로 안 망가짐)
- 원칙: **지속가능한 일관성 > 단기 완벽**. (책·후추가루 둘 다 극단보다 일관성)
