# ADHD 보상계 회복 — 도파민 내성·하향조절

> 출처: ADHD 도파민 트윗(@ADzzzzshd) 리뷰, 2026-06. 후추가루 노선은 아니지만 게임·각성·영양과 직결.
> 연계: [`relaxed-alertness.md`](./relaxed-alertness.md), [`dark-calories.md`](./dark-calories.md), [`hypoglycemia-bridge-protocol.md`](./hypoglycemia-bridge-protocol.md)
> 핵심 모델: 보상계는 "망가진" 게 아니라 **"강자극에 둔감해진(하향조절)"** 것. 강자극을 줄이면 천천히 회복.

---

## 0. 한 줄 요약
ADHD 뇌 = **기저 도파민이 낮음** → 자극 추구. 강한 보상(SNS·술·도박)을 쉽게·대량으로 반복하면
**수용체가 둔감해져(하향조절)** 일상 자극엔 무감각 → 무기력·무쾌감·우울.
회복법 = **강자극 줄이고 저자극 보상을 꾸준히** → 수용체 민감도가 시간을 두고 회복.

---

## 1. ✅ 트윗에서 맞는 부분 (실제 신경과학)
- ADHD = **낮은 tonic 도파민** → 강자극엔 과반응(과집중·중독). "낮은 베이스라인 + 강자극 과반응" 정확.
- 강하고·쉽고·대량인 도파민(SNS·술·도박) → **초생리적 폭발 반복 → 수용체 둔감화.** 잘 입증된 메커니즘(『도파민네이션』류).
- 결과 = 무기력·무쾌감(anhedonia)·우울 → 하향조절/금단 상태와 일치.
- 해결: **디지털 디톡스 + 금주 + 저자극 활동(산책·독서) 지속** → 민감도 회복 시간 벌기. 방향 옳음.

## 2. ⚠️ 빠졌거나 과장된 부분 (걸러 들을 것)
- **"고칠 수 있다"는 과장.** ADHD는 유전·신경발달 기반 → *개선*은 되나 *완치*는 아님. "망가진 걸 고친다"보다 **"둔감해진 민감도를 회복"**이 정확.
- **약물(자극제) 언급 없음.** 일부 ADHD엔 약물치료가 진짜 효과적. 행동요법만 만능처럼 읽으면 위험.
- **"도파민 디톡스" 용어는 부정확.** 도파민을 빼는 게 아니라 **초자극 입력을 줄이는** 것. 실천은 옳지만 용어 오해 주의.
- **시간 걸림 + 우울은 임상 영역.** 회복은 수 주~수 개월. 무기력·우울 심하면 전문가 상담 병행.

## 3. 🔗 나에게 적용 (핵심)
- **게임 = 이중 타격:** 강한 도파민(보상계 둔감화) + 투쟁-도피 각성(relaxed-alertness 참조).
  → 게임 후 더 무기력하고 일상이 시시해지는 이유. 두 메커니즘이 겹침.
- **저자극 활동(산책·독서) = relaxed alertness / 부교감 회복과 같은 방향.** 보상계 회복 + 각성 진정 동시.
- **영양이 보상계를 받친다:**
  - 도파민 합성엔 **철분(페리틴!)·티로신·B군** 필요. (→ 페리틴 검사, 레버·적색육)
  - **안정된 혈당** = 기분·도파민 고르게. 저혈당 크래시는 도파민 회복 방해.
  - 즉 **우리 식단 노선이 도파민 회복의 토대.**

## 4. 🎯 실천 세트 (한 묶음으로 작동)
1. **강자극 줄이기:** 게임·SNS·도박·술 — 빈도·강도·접근성 낮추기.
2. **저자극 보상 꾸준히:** 산책, 독서, 햇빛, 단순노동. (당장 시시해도 그게 회복의 신호)
3. **각성 진정:** 힘 빼기, 느린 호흡 (relaxed-alertness 도구).
4. **영양 토대:** 페리틴 확인, 혈당 안정(당+단백질+소금), 전유·레버.
5. **시간 주기:** 수 주~수 개월. 무쾌감은 회복 중간 단계지 실패 아님.

> 판정: 트윗 신뢰도 7/6.5 (상위권). "완치 뉘앙스 + 약물 누락 + 디톡스 용어"만 걸러 들으면 실천 가이드로 훌륭.
> 한 줄: 보상계는 망가진 게 아니라 **둔감해진** 것 → **게임 줄이기 + 산책·독서 + 철분·혈당 안정**이 한 세트.
